# VisuTerreImac
Moteur de rendu 3d pour terrain a partir d''une heightmap


## Pour compiler : 

* make main
* bin/main.out

## Touches :
* Avancer: Z
* Reculer : S
* Droite : D
* Gauche : Q
* Bouger la caméra avec la souris
* Monter : fleche haut
* Monter : fleche bas
* Augmenter LOD : M
* Diminuer LOD : L
* Augmenter hauteur du point visé : Y
* Quitter : Esc
